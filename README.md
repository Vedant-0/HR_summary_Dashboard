# HR_summary_Dashboard

## 1) Project Title / Headline

HR Summary Dashboard for Workforce Insights & Analytics


## 2) Short Description / Purpose

This project analyzes HR data to monitor employee trends, workforce distribution, and key HR metrics. 

The goal is to support data-driven HR decisions by tracking employee performance, attrition, hiring trends, and department-wise insights.


## 3) Tech Stack
📊 Power BI (Dashboard & Data Visualization)

🐍 Python (optional for preprocessing)

📁 Excel / CSV (Data Handling)

📈 DAX (Measures & Calculations)


## 5) Data Source

The dataset used in this project is based on HR records, including employee details, job roles, salaries, departments, and attrition data.

Data includes:
Employee demographics
Department & job roles
Salary & performance metrics
Attrition / retention data

Note: This dataset may be simulated or sourced from public datasets (e.g., Kaggle) for learning purposes.

## 6) Features / Highlights

📌 Interactive HR Dashboard

👥 Employee Distribution by Department & Role

📉 Attrition Analysis (Who is leaving & why)

💰 Salary & Compensation Insights

📊 Performance & Experience Trends

📅 Hiring & Workforce Growth Overview

🔍 Filters for Department, Gender, Role, etc.


## Business Questions Studied

Which department has the highest attrition?

What is the overall employee retention rate?

How does salary impact employee attrition?

Which job roles have the highest turnover?

What is the gender distribution across departments?

How does experience affect performance?


## Key Questions Answered

Which departments need HR intervention?

Where is employee attrition highest?

What trends are visible in hiring and exits?

Which roles are most stable/unstable?

How can HR improve retention strategies?
